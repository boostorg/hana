Detailed Description

template<typename M, typename Enable = void>
struct boost::hana::Monad< M, Enable >

Monads are Functors with the ability to wrap values into them and flatten values that were wrapped more than once.

Minimal complete definition

unit and (join or bind)

Laws

Instances of Monad must satisfy the following laws:

bind(unit<M>(x), f) == f(x)
bind(m, unit<M>) == m
bind(m, [](auto x){ return bind(f(x), g); }) == bind(bind(m, f), g)
